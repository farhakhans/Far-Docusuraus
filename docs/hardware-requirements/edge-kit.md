# Edge Kit

The Edge Kit offers a more capable platform for advanced robotics projects, featuring increased computational power and enhanced sensing capabilities while maintaining affordability.

## Specifications

### Computing Platform
- NVIDIA Jetson Orin Nano Developer Kit (4GB or 8GB RAM)
- Alternative: NVIDIA Jetson AGX Orin Developer Kit (for maximum performance)
- Storage: 64GB eMMC + microSD slot for expansion
- Connectivity: Wi-Fi 6, Bluetooth 5.0, Ethernet

### Enhanced Sensors
- Stereo Camera: Dual RGB cameras with depth estimation capability
- IMU: High-precision 9-axis IMU (accelerometer, gyroscope, magnetometer)
- LiDAR: 2D LiDAR sensor for mapping and navigation (optional upgrade)
- Force/Torque Sensors: Integrated at key joints for tactile feedback
- Environmental Sensors: Temperature, humidity, barometric pressure

### Actuation & Control
- High-Torque Servos: 16+ digital servos with feedback
- Custom Motor Controllers: PID-controlled driver boards
- Redundant Power System: Primary and backup battery packs
- Power Management: Smart power distribution and monitoring

### Structural Components
- Machined aluminum frame elements for durability
- Carbon fiber structural components
- Precision-machined joints with backlash reduction
- Modular design for easy maintenance and upgrades

## Included Software
- Full ROS2 ecosystem with advanced packages
- NVIDIA Isaac ROS acceleration packages
- Pre-trained neural networks for perception tasks
- Advanced simulation environments
- Cloud connectivity options

## Advantages
- Sufficient compute for real-time AI inference
- Professional-grade sensor suite
- Robust construction for extended use
- Extensible architecture for custom additions
- Industrial-grade components
