# Economy Kit

The Economy Kit provides an affordable entry point into the Physical AI Humanoid Robotics program, designed for individual learners, small groups, or institutions with limited budgets.

## Specifications

### Computing Platform
- Single Board Computer (SBC): Raspberry Pi 4 Model B (8GB RAM)
- Alternative: NVIDIA Jetson Nano Developer Kit
- Storage: 64GB microSD card (Class 10 or higher)

### Basic Sensors
- RGB Camera: 5MP or higher resolution
- IMU (Inertial Measurement Unit): MPU-6050 or equivalent
- Distance Sensor: Ultrasonic sensor (HC-SR04) or Time-of-flight (VL53L0X)
- Encoders: Magnetic rotary encoders for joint position feedback

### Actuation
- Servo Motors: 12-16 standard servo motors (SG90 or equivalent)
- Motor Controller: PWM controller board compatible with selected SBC
- Battery: Rechargeable LiPo battery pack (7.4V, 2200mAh minimum)

### Structural Components
- 3D printed frame components (PLA or ABS filament)
- Fasteners and hardware kit
- Wiring harness and connectors

## Included Software
- Pre-configured ROS2 installation
- Basic control interfaces
- Simulation environment access
- Documentation and tutorials

## Advantages
- Cost-effective solution under $500
- Suitable for learning fundamental concepts
- Easy to maintain and replace components
- Compatible with extensive online resources

## Limitations
- Limited computational power for complex AI tasks
- Reduced payload capacity
- Basic sensor suite
- May require additional components for advanced projects

## Recommended Use Cases
- Individual student projects
- Small classroom deployments
- Proof-of-concept demonstrations
- Basic robotics education

## Assembly Requirements
- Basic electronics knowledge
- 3D printing capabilities (or access to service)
- Standard hand tools
- Estimated assembly time: 10-15 hours
